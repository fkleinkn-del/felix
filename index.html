The problem

GA4 debugging is noisy, slow,
and scattered across tabs.

Felix Debugger pulls every event, parameter and consent signal into one calm, readable timeline, right inside your DevTools.

Features

Everything you need.
Nothing you don't.

Real-time debugging

Watch GA4 events stream in the moment they fire. No refresh, no guesswork.

Timeline & insights

A clean, chronological view of every interaction with parameters, user properties and context.

Validation & QA

Catch missing parameters, broken naming and invalid values before they reach production.

Event comparison

Diff two events side-by-side. Spot inconsistencies between staging and production in seconds.

E-commerce ready

Inspect items, currencies, transaction IDs and the full purchase funnel, fully expanded.

Advanced tools

Filter, search and export. Inspect Consent Mode signals and page performance in the same view.

Built for clarity

Every event.
Decoded beautifully.

Parameters, user properties, consent state and timing, laid out in a way that makes sense at a glance.

How it works

Three steps. That's it.

  1. 01

    Install the extension

    One click from the Chrome Web Store. No account, no setup.

  2. 02

    Open DevTools

    Find the Felix Debugger panel right next to Console and Network.

  3. 03

    Debug instantly

    Trigger any event and watch it decoded, validated and timestamped. Live.
