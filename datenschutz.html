2. Data processing principles

We process personal data only when necessary to provide a functioning website and its content. Processing is based on a legal permission or explicit consent pursuant to Art. 6 GDPR.

3. Data processing by the Chrome extension

The Felix Debugger browser extension processes data exclusively locally on your device.

  • No personal data is collected, stored or transmitted.
  • No tracking or usage analysis takes place.
  • No data is sent to external servers or third parties.
  • The extension only accesses GA4 network requests from the currently open tab to display them in the DevTools debug panel. This processing is entirely local in your browser.
  • Inputs (e.g. filters, comparison data) remain in your local browser storage and are not synced.

No personal data is transmitted to us through use of the extension.

4. Website hosting

This website is hosted by a third-party provider. When the site is accessed, technically necessary information (server log files) is recorded by the hosting provider:

  • Anonymised IP address
  • Date and time of the request
  • Data transferred and HTTP status code
  • Requesting browser, operating system and referrer

Processing is based on our legitimate interest in stable and secure operation of the website (Art. 6 para. 1 lit. f GDPR). Data is not merged with other data sources.
